# 修改日志（2026-03-30）

## 版本主题
首页导航可配置能力增强 + 原型模板库工具页上线。

## 主要改动

### 1. 首页导航能力增强
- 支持大类、小类、入口的可视化管理。
- 支持入口删除与找回（已删除入口可批量恢复）。
- 支持双击重命名：
  - 大类
  - 小类
  - 入口
- 支持拖动排序与迁移：
  - 大类拖动排序（整组跟随）
  - 小类拖动排序/跨大类迁移（入口跟随）
  - 入口拖动排序/跨小类迁移
- 优化拖动交互：支持中间位置插入，新增清晰落点高亮。
- 左侧导航视觉调整：
  - 标题不显示数量
  - 默认白字
  - 当前页面高亮为橘黄色

### 2. 模板库工具页（MVP）
- 新增路由：`/tools/template-library`
- 新增页面：原型模板库
  - 模板分类筛选（表格/列表/审批/弹窗）
  - 模板卡片列表
  - 样式变量面板（颜色、字号、圆角、紧凑模式）
  - 实时预览
  - 调用口令生成与复制

### 3. 导航与发布限制策略调整
- 模板库路由 `\/tools\/template-library` 在发布白名单模式下也可访问。
- 首页导航数据源改为全量路由，确保新增页面自动进入导航入口，不依赖手动配置。
